# 学术写作规范

## 1. 语言风格

- 使用正式、客观的学术语言
- 避免主观臆断和情绪化表达
- 使用第三人称叙述

## 2. 结构要求

### 2.1 引言
- 研究背景
- 研究现状
- 研究空白
- 研究目的

### 2.2 方法
- 研究对象
- 研究设计
- 数据收集
- 数据分析

### 2.3 结果
- 描述性统计
- 主要结果
- 次要结果

### 2.4 讨论
- 主要发现总结
- 与已有研究的比较
- 局限性
- 结论与展望

### 2.5 结论
- 核心发现总结
- 理论和/或实践意义
- 未来研究方向

## 3. 引用规范

- 使用 GB/T 7714 格式
- 所有断言需有引用支撑
- 引用必须来自真实存在的文献

## 4. 常见问题

### 4.1 避免虚假引用
- 不要编造不存在的文献
- 不要编造不存在的作者
- 使用学术数据库核实文献信息

### 4.2 避免抄袭
- 合理引用他人成果
- 注明引用来源
- 保持原创性
