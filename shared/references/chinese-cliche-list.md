# 中文论文写作反面案例库

本文档列出常见的 AI 生成中文论文的典型问题，供 `paper-writer` 和 `paper-polisher` 在写作和润色时参照。

## 一、翻译腔问题

| 错误模式 | 错误示例 | 正确示例 | 问题说明 |
|---------|---------|---------|---------|
| "进行/做出/给予" + 抽象名词 | 进行了一个实验 | 开展了实验 | 中文应使用具体动词 |
| | 做出了重要贡献 | 做出了重要贡献（可接受，但可改为"贡献突出"） | 减少冗余 |
| | 给予了充分考虑 | 充分考虑 | 直接使用动词 |
| "在这种情况下" | 在这种情况下，我们发现... | 在此背景下，研究发现... | "在这种情况下"是典型的英文 "in this case" 直译 |
| "值得注意的是" | 值得注意的是，结果... | 结果表明... | 冗余表达，直接陈述即可 |
| "基于以上分析" | 基于以上分析，可以得出... | 综上... | "基于"在英文中是 "based on" 的直译 |
| "关于...的研究" | 关于X对Y影响的研究 | X对Y的影响研究 | 去掉"关于"更简洁 |

## 二、过度连接词

| 问题 | 说明 |
|------|------|
| "此外"高频出现 | 每段开头都用"此外"，显得机械。应通过逻辑自然衔接 |
| "与此同时" | 每段过渡都使用，应减少频率 |
| "综上所述" | 一篇论文中最多使用1次，通常在讨论末段 |
| "首先...其次...最后" | 过度使用列表式连接，使行文生硬 |

## 三、空洞/夸大表述

| 错误示例 | 问题 | 修正方向 |
|---------|------|---------|
| 本研究具有重要的理论和实践意义 | 空洞，未说明具体意义 | 具体说明对哪个领域/人群有什么意义 |
| 取得了较好的效果 | "较好"是模糊表述 | 报告具体数据和统计结果 |
| 为...提供了有力支撑 | 过度评价 | 说明具体如何支撑 |
| 填补了该领域的空白 | 过度夸大 | 说明具体填补了哪个方面的空白 |
| 首次证明了... | 无法验证 | 改为"本研究表明..." |
| 重大突破 / 开创性 / 革命性 | 过度夸大 | 使用客观表述 |

## 四、统计数据表述问题

| 错误示例 | 问题 | 正确示例 |
|---------|------|---------|
| 数据表明某种趋势 | 模糊限定（可能 p>0.05） | 数据表明X显著高于Y（p<0.001） |
| 差异不显著 | 未报告具体 p 值 | 差异无统计学意义（p=0.12） |
| 有统计学意义 | 未报告效应量 | 有统计学意义（d=0.85, p<0.001） |
| 结果一致表明 | 过度概括 | "X研究...表明...；Y研究...表明..."分别说明 |

## 五、逻辑谬误模式

| 谬误类型 | 错误模式 | 识别特征 |
|---------|---------|---------|
| 相关→因果 | "A导致B"但研究设计为观察性 | 检查研究设计是否为 RCT |
| 选择性报告 | 只讨论显著结果 | 检查是否有不显著结果被忽略 |
| 事后假设 | 结果→假设→讨论 | 检查假设是否在 Methods 前就已提出 |
| 过度推断 | p=0.049→"显著差异"，p=0.051→"无差异" | 避免 p 值边界处的不对称解读 |
| 生态谬误 | 群体水平结论→个体水平推论 | 注意研究层面和推论层面的一致性 |

## 六、段落结构问题

| 问题 | 说明 |
|------|------|
| 段落过长 | 单段超过 300 字，应拆分为多个段落 |
| 段落无主题句 | 每段应有明确的主题句，放在段首 |
| 段落间无过渡 | 相邻段落跳跃感强，缺乏逻辑衔接 |
| 论点→论据不对应 | 段落主题句与后续内容不匹配 |
